Configurar Liberação de Ofertas

Defina a data e período a partir do qual as solicitações de ofertas estarão disponíveis.

A data e o período selecionados determinam o ponto de partida para novas solicitações. As regras são aplicadas no dia escolhido, e os dias seguintes são liberados normalmente.

A data é obrigatória.
